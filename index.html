v-show 和 v-if 区别

  • v-show 通过 CSS display 控制显示和隐藏
  • v-if 通过判断组件真实渲染和销毁，而不是显示和隐藏
  • 频繁切换显示状态用 v-show，否则用 v-if

v-if

  • 当 v-if 与 v-for 一起使用时，v-for 具有比 v-if 更高的优先级，意味着：v-if 将分别重复运行于每个 v-for 循环中，会造成性能问题。所以，不推荐 v-if 和 v-for 同时使用
